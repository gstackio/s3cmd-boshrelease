s3cmd BOSH Release
==================

This BOSH Relase provides two errands `create-buckets` and `remove-buckets` in
order to create or destro S3 buckets, using the [`s3cmd`][s3cmd_repo] CLI.

[s3cmd_repo]: https://github.com/s3tools/s3cmd
